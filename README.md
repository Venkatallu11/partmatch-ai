## What is PartMatch AI?

PartMatch AI is a free global industrial parts search engine built for maintenance engineers, procurement managers, and repair shops worldwide.

When a machine breaks down and you need a hydraulic seal, a bearing, a bolt, or any industrial part — PartMatch AI finds the best suppliers globally in seconds, comparing prices from China, India, USA, Europe, Africa and the Middle East.

---

## The Problem It Solves

Finding aftermarket industrial parts is painful:
- Part numbers don't match across brands
- Supplier catalogs are outdated PDFs
- You call 5 distributors and wait 2 days
- A machine sitting idle costs $15,000/day

PartMatch AI fixes this.

## Features

### 🧠 4-Layer Intelligent Search

| Layer | What it does | Speed |
|-------|-------------|-------|
| 1. Smart Parser | Detects part type, material, brand, dimensions | Instant |
| 2. Supplier Directory | 37 verified global suppliers | Instant |
| 3. Live Web Search | Brave API finds real listings | 2–3 sec |
| 4. AI Analysis | OEM numbers, alternatives, pricing | 5–10 sec |

## How to Use

1. Go to **[partmatch-ai.vercel.app](https://partmatch-ai.vercel.app)**
2. Type any part description:
   - `"M8 hex bolt grade 8.8 stainless"`
   - `"80mm piston seal NBR JCB excavator"`
   - `"6205 deep groove ball bearing NSK"`
   - `"DN50 ball valve stainless PN16"`
3. See results from 37 global suppliers instantly
4. Add a free Brave API key for live web prices
5. Add an AI key (Claude / GPT-4o / Gemini) for expert part analysis
